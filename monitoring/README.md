# Netdata

Real-time system monitoring with per-second granularity.

## Overview

Netdata provides instant visibility into system performance with minimal configuration. Complements the main monitoring stack in [opensense-monitoring](../opensense-monitoring/).

## Features

- Per-second metric granularity
- Zero configuration auto-detection
- Low resource usage
- Built-in visualizations

## Deployment
```bash
docker-compose up -d
# Access at http://host:19999
```
